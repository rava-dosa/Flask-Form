Department of Computer Science & Engineering, I.I.T. Kharagpur organizes a short term course entitled “ C++ Programming and Data Structure ” from 27th June to 5th July, 2017.

Aims & Objectives :

This course aims to build over previous knowleadge of C to illustrate its shortcommings and how these are overcome in C++.
Next it introduces object and class concepts and treats each data structure like Array, Record, Stack, Queue, Lists, Trees as either composed of several classes and / or derived from some classes alongwith the relevant methods, which represents an unique treatment.

Who can participate?

Category-1 (Students) : At least 1st year undergraduate pass out with formal coursework in C.

Category-2 (Others) : Teachers / Scientific Officers / Instructors / Technical Assistants / Persons from Industries.
